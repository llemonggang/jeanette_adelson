  • Tic-Tac-Toe

    was my first challenge with JavaScript, and a huge lessons in writing functions, as well as learning to conceptulaize an entire project before beginning to code. Using a basic HTML template, I wrote functions using JavaScript and the JQuery library. When the page loads, Player 1 is the first player to go, and when they click an empty cell, it is marked with an "X", then the Current Player switches to Player 2. When Player 2 clicks an empty cell, it is marked with an "O", and then the Current Player switches back to one. If a player clicks a cell that has already been marked, they see an alert letting them know that the cell in question has already been taken. When either player gets three in a row, they are pronounced the winner and the game resets with the opposite player starting the game.
